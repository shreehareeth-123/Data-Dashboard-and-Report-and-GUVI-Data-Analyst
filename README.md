# Data Dashboard and Report

## Project Overview
This project analyzes two months of simulated sales data and creates an Excel-based dashboard (pivot tables and charts).

## Files in this repo
- `Sales_Dashboard.xlsx` — Raw simulated sales data (generated by generate_sales_data.py)
- `Sales_Dashboard_with_Charts.xlsx` — Workbook with Pivot_Charts sheet (generated by add_pivot_charts.py)
- `generate_sales_data.py` — Script to generate sample data
- `add_pivot_charts.py` — Script to create summary tables and charts in Excel
- `generate_readme_pdf.py` — Script to create the README PDF
- `README_Data_Dashboard.pdf` — PDF version of this README (optional)

## How to run
1. Ensure Python 3.8+ is installed.
2. Install required packages:
   ```bash
   pip install pandas numpy openpyxl reportlab
   ```
3. Run:
   ```bash
   python generate_sales_data.py
   python add_pivot_charts.py
   python generate_readme_pdf.py  # optional
   ```

## Notes
- The scripts will create the Excel files in the working directory.
- If you want to customize the dataset (more rows, different date range), modify `generate_sales_data.py`.
